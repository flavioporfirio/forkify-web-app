## Features

- Search for recipes; 
- Add recipes;
- Change servings; and
- Bookmarks.

## How To Use

```bash
# Clone this repository
$ git clone https://github.com/flavioporfirio/forkify-web-app

# Install dependencies
$ npm install

# Run the app
$ npm run start
```
